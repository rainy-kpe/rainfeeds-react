# rainfeeds-react
React version of Rainfeeds

# Prerequisites
* Install Node
* Install Yarn (`npm install -g yarn`)

# Scripts
* `yarn build` Builds the release version and places the files to `dist` folder
* `yarn start` Builds the development version and runs the dev server
